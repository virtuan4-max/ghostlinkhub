GHOSTLINK INTERFACE

WARNING : DO NOT DISCLOSE THE WEBSITE, API KEYS, OR ANY OTHER PROPERTY RELATING TO PROJECT BLUE PINNAPPLE.

OVERVIEW: GhostLink is a terminal-style AI interface powered by the Groq API.

USAGE: This website uses cloud-managed API keys fetched from a secure vault and rotated automatically. You can optionally supply your own key in Settings.

OPERATIONAL LOGIC:
• Neural Link: Connects to cloud-based LLMs via direct API calls to Groq.
• Memory: Maintains a volatile 'currentHistory' array in local RAM.
• Multimodal: Supports up to 5 simultaneous image uploads via Base64 encoding.
• Key Rotation: Cloud keys rotate every message to distribute load and avoid limits.
• Chat History: Sessions saved to localStorage, click the clock icon to browse.
• Interface: Single-file client-side architecture.

SECURITY: Encrypted end-to-end. Session data exists only in local RAM and is purged upon manual wipe or browser refresh.
